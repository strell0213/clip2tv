# Clip2TV

Flutter приложение для Android TV, которое позволяет отправлять текст с телефона на телевизор через локальную сеть.

## Описание

Приложение поднимает локальный HTTP-сервер на Android TV, генерирует QR-код с URL сервера и отображает его на экране. Пользователь сканирует QR-код с телефона, открывает локальный сайт в браузере, вводит текст и отправляет его. Текст автоматически копируется в буфер обмена телевизора.

## Архитектура

Проект использует Clean Architecture:

- **domain/** - бизнес-логика (entities, repositories, use cases)
- **data/** - реализация репозиториев и data sources
- **presentation/** - UI слой (pages, providers)

## Функциональность

- ✅ Локальный HTTP-сервер на Android TV
- ✅ Генерация QR-кода с URL сервера
- ✅ Получение локального IP адреса
- ✅ Красивый веб-интерфейс для ввода текста
- ✅ Автоматическое копирование текста в буфер обмена
- ✅ Работа без интернета (только локальная сеть)
- ✅ Оптимизация для Android TV (навигация с D-pad)

## Установка

1. Установите зависимости:
```bash
flutter pub get
```

2. Запустите приложение на Android TV:
```bash
flutter run
```

## Использование

1. Запустите приложение на Android TV
2. Приложение автоматически запустит сервер и отобразит QR-код
3. Отсканируйте QR-код с телефона
4. Введите текст в браузере и нажмите "Отправить"
5. Текст будет скопирован в буфер обмена телевизора

## Технологии

- Flutter/Dart
- `dart:io` HttpServer
- `qr_flutter` - генерация QR-кодов
- `clipboard` - работа с буфером обмена
- `provider` - управление состоянием

## Требования

- Android TV устройство
- Телефон и телевизор должны быть в одной локальной сети
- Разрешения: INTERNET, ACCESS_NETWORK_STATE
